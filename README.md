# Arctic Mule (Model BWB-1)

## Tagline
The World's First Hybrid-Electric Bush Freighter.

## The Hook
"Electric aviation failed the North. We fixed it."

## Core Technology
- **Airframe:** High-Lift Blended Wing Body (BWB). 2-Pallet wide cargo bay.
- **Power:** "Power Pallet" System. A modular, slide-in turbogenerator (Jet-A/Diesel) that turns the plane from EV to Hybrid in 5 minutes.
- **Battery:** Sodium-Ion Chemistry. Survives -40Â°C "Cold Soaks" without permanent damage.
- **Operation:** "Snowbird" Capable. Swap to Lithium packs for Caribbean summer operations.

## Key Specs
- **Payload:** 1,200 kg (2,650 lbs) / 2 Standard Pallets.
- **Range:** 600 nm (Hybrid) / 200 nm (Pure Electric).
- **Runway:** 2,500 ft (Gravel/Snow Capable).
- **Speed:** 180 knots cruise.

## Target Market
Northern Canadian supply chains (Yellowknife, Nunavut) and Caribbean Island hoppers.
